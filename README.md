# gtm-cowork-vs-code

**Claude Cowork vs Claude Code for B2B Sales and Marketing Teams** — a single-page
guide by **Jon Russo / B2B Fusion**, hosted on Vercel with auto-deploy from this repo.

## Contents

- `index.html` — the full page (self-contained; all CSS inline)

## Editing & deploying

Edit `index.html`, then push to `main` — Vercel redeploys automatically.

```bash
git add -A && git commit -m "your message" && git push origin main
```

Preview locally:

```bash
python3 -m http.server 8000   # then visit http://localhost:8000
```
